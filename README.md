# Sales Dashboard Excel Project

## Description
This project is a Sales Dashboard created using Microsoft Excel to analyze and visualize sales data.  
The dashboard helps to track sales performance by category, product, and date using charts, pivot tables, and slicers.

## Features
- Sales by Category analysis
- Sales by Product visualization
- Sales by Date tracking
- Interactive Date Slicer
- Pivot Tables
- Charts and Graphs
- Easy sales performance analysis

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers

## Dashboard Insights
- Electronics category has the highest sales
- Laptop product generated maximum revenue
- Dashboard allows quick filtering by date
- Sales trends can be analyzed visually
